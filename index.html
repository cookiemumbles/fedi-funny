Funny twitter friends to follow on Mastodon

How to:

  1. Select some or all below
  2. Click "Get CSV" and download the file.
  3. Go to your mastodon profile in a browser
  4. [Preferences] > [Import and Export] > [Import]
  5. Select [Following list] > [Browse] > select file > [Upload]

You can also check out this instructions pdf with pictures.
